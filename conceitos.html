História da Internet

A Internet como conhecemos foi concebida incialmente após os telegrafos. Os computadores na medida em que foram se desenvolvendo foi necessário criar uma rede onde eles se comunicassem.

Com o inicio da guerra fria, tanto a URSS como os EUA investiram em maquinas que pudessem otimizar os caculos de guerra. Sendo assim tem=se como primeira rede de comunicação entre maquinas a arpanet em 1969.

O que é HTML !?

O que são Servers ?

São os sistemas onde ficam armazenados as instruções e dados que os Clients irão acessar. Lá é onde consta os códigos, bancos de dados entre outro conjunto de arquivos que os clientes irão acessar.

Quais os tipos de Linguagens existentes?

Existem diversos tipos de forma de escrever projetos com forma de codigo diferente. Devido a popularização de algumas empresas multinacionais, o JAVAScript é a mais utilizida dentre elas. Mas temos também a Phyton, C++, Rubi e outras por exemplo.

O que são tags ?

"Tags" são as formas de indicacao no editor de texto qual o tipo de função que irá ser realizada, por exempo a "TAG p" indica que iremos escrever um paragráfo. Existem outras que também são de suma importância que veremos no decorrer do aprendizado.
